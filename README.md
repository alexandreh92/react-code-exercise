## Setup
- To run without docker just run:

  ```sh
  yarn install
  yarn start
  ```

- To run with docker:

  ```sh
  docker-compose up -d --build
  ```

  and open your browser in `http://localhost:3001`

## Lint

To run linters just do:

```sh
yarn lint
```

## Tests

To run tests:

```sh
yarn test
```
